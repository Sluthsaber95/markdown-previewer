# Markdown Previewer
This project was bootstrapped with [Create React App]
(https://github.com/facebookincubator/create-react-app).

## Getting Started
Fork project from Github, and install all modules with the project
```
npm i
```

Or if you are using Yarn.
```
yarn i
```
